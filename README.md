# jctec-lib

Biblioteca pessoal para o **KiCad**, criada para organizar e reutilizar símbolos, footprints, modelos 3D e componentes usados nos projetos eletrônicos da JCTec.

Este repositório tem como objetivo centralizar os arquivos de biblioteca utilizados em projetos de hardware, facilitando manutenção, versionamento e reaproveitamento entre diferentes placas.

## Objetivo

Manter uma biblioteca própria e versionada para projetos desenvolvidos no KiCad, incluindo:

- Símbolos esquemáticos customizados
- Footprints personalizados
- Modelos 3D
- Componentes revisados e validados
- Bibliotecas específicas para módulos, conectores, placas e circuitos usados nos projetos
